Vista'da kısayol oklarından kurtulmak

Sizi bilmem ama ben kısayolların sol altındaki okları sevmem. Bilemeyecek miyim hangisi kısayol hangisi asıl program? Batar bana orda o ok. Eğer siz de böyle düşünüyorsanız Windows Vista’da bu ok nasıl kaldırılır görelim.

Öncelikle okun yerini alacak simgeyi belirlemeniz gerek. Yerini alacak simge “Boşluk” olursa ok görünmez. “Boş” ikonu buradan indirebilirsiniz. Zipin içinden çıkacak ikonu istediğiniz bir yere atın. (Örnek: C:\Windows\blank.ico) Daha sonra aşağıdakileri adım adım uygulayabilirsiniz.

 

  1. Başlat menüsüne gelin.
  2. Arama çubuğuna “regedit.exe” yazarak ENTER’e basın.
  3. HKEY_LOCAL_MACHINE \ SOFTWARE \ Microsoft \ Windows \ CurrentVersion \ explorer \ Shell Icons ağacına gidin.
  4. Sağ tarafta sağ tıklayarak 29 isimli yeni bir String girdisi oluşturun. (REG_SZ)
  5. 29’a çift tıklayın ve içeriğini (Value) ikonun yolu olarak değiştirin. (Örnek: C:\Windows\blank.ico)
  6. Bilgisayarı yeniden başlatın. (Eğer uzman bir kullanıcıysanız: bilgisayarı kapatmadan explorer.exe yi yeniden başlatırsanız da değişiklikler etkin olabilir.)

Yararlanılan Kaynak: http://www.winhelponline.com/articles/153/1/
