MSDNAA’dan İndirdiğiniz Language Pack ile Windows 7’yi Türkçe Yapmak

{right}Öğrencisiniz. Üniversiteniz MSDN Academic Alliance üyesi ve size Microsoft’un bazı ürünlerini ücretsiz sağlıyor.

Bunların içinde Windows 7 Professional de var. Ama İngilizce. Siz ise, işletim sisteminizi İngilizce kullanmak istemiyor, Türkçe olsun istiyorsunuz.

MSDNAA’da Windows 7 Language Pack diye bir şey daha var. İndirdiniz onu da. Bir DVD boyutundaydı!

İndirdikten sonra açtınız/DVD’ye yazdınız ama o da ne! Setup dosyası falan yok! Windows 7 Professional yüklü, elinizde İnternet’ten indirdiğiniz MSDNAA Windows 7 Language Pack var, ama kuramıyorsunuz.

O zaman aşağıdaki adımları izleyin. =)

Gerekenler: Windows 7 Professional, Windows 7 Language Pack (MUI Dil Paketi)

İkisi de lisanslı ve ücretsiz olarak varsa MSDNAA hesabınızdan indirebileceğiniz paketlerdir. =) Dikkat etmeniz gereken, indirdiğiniz dil paketi ile işletim sisteminizin biti birbirine uymalıdır. 32-bit işletim sistemi kurduysanız x86, 64-bit işletim sistemi kurduysanız x64’ü indirmiş olmalısınız.

Adım Adım Kurulum

Adım #1

Öncelikle yönetici olarak komut istemini açın. Bunu yapabilmenin en kolay yolu, başlat menüsünü açmak ve en alttaki arama kutucuğuna CMD yazmak. Yukarıdaki sonuçlarda cmd.exe göründüğünde, cmd.exe'ye sağ tıklamak ve Run as Administrator'u seçmek.

Adım #2

Daha sonra aşağıdaki komutu, kendinize göre düzenleyerek çalıştırın:

DISM /Online /Add-Package /PackagePath:(DİL PAKETİNİN YOLU) Örnek: DISM /Online /Add-Package /PackagePath:*E:\languagepack\tr-tr*

Adım #3

Sonra aşağıdaki komutu kullanın: bcdedit /set {current} locale tr-TR

Adım #4

Ve daha sonra şu komutu kullanın: bcdboot %WinDir% /l tr-TR

Adım #5

Son olarak registryi açın. Başlat menüsündeki kutucuğa regedit.exe yazarak açabilirsiniz.

Aşağıdaki ağaca gidin: HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\MUI\UILanguages

Adım #6

Ağaçlardan en-US isimli olanını silin.

Adım #7

Bilgisayarınızı yeniden başlatın. Bilgisayarınız Türkçe olacak.
