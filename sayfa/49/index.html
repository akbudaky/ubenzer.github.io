Microsoft Yaz Okulu 2010 İzmir Başladı

 

Microsoft Yaz Okulu 2010’un İzmir ayağı 2 Ağustos 2010 Pazartesi günü 9 Eylül Üniversitesi’nde başladı. Aynı zamanda stajım olarak da kabul edilecek bu etkinlikte, Microsoft’un birçok teknolojisi hakkında bilgi sahibi olacağız.

Pazartesi günü Muammer Benzeş hocamızla başladığımız Windows Server 2008 R2 eğitimine bugün Bilge Adam’dan Hakan Ulagan hocamızın anlatımında SQL Server 2008 R2 ile devam ettik.

Eğitimler verimli geçiyor. Zaten daha önceden az da olsa bildiğimiz şeyleri, uzmanlarından öğrendiğimiz ince ayrıntılarla pekiştirince ortaya güzel bir şeyler çıkacağına inanıyorum.
