Türkiye'deki Şehirler SQL

Bugün yazdığım programın bir yerinde Türkiye’deki şehirleri veritabanından çekmem ve bir yerde listeletmem gerekti. Programlama kısmını yapmak sorun olmadı ama Google’da şöylesine bir arattırdıysam da SQL formatında saklanmış bir şehir listesi bulamadığımdan şehirleri veritabanına aktarmak kolay olmadı.

Ben de hazır kendi veritabanıma eklemişken, bu şehirler tablosunu sizinle de paylaşayım dedim. Eğer bir gün şehirleri uygulamanızda kullanmanız gerekirse aşağıda bulabileceğiniz SQL’i düzenleyip çalıştırmanız yetecektir.

ID -> Birincil anahtardır, şehrin plakasına göre belirlenir. name -> Şehrin adıdır.

Diğer bilgilere ihtiyacım olmadığı için başka alan ekleme gereği duymadım. SQL cümleciklerini aşağıda görebilir ya da buradan bilgisayarınıza indirebilirsiniz.

CREATE TABLE IF NOT EXISTS sehirler ( ID int(3) unsigned NOT NULL default '0' COMMENT 'Plaka Numarası', name tinytext NOT NULL, PRIMARY KEY (ID) ) ENGINE=MyISAM DEFAULT CHARSET=utf8;
