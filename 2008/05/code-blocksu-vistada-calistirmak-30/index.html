Code Blocks’u Vista’da Çalıştırmak 3.0

Daha önce buradaki ve buradaki yazılarımda Code Blocks’u Vista’da çalıştırma yöntemlerini anlatmıştım. Ancak gel zaman git zaman buradaki yöntemler de eskidi.

Yöntem 2’ye artık ihtiyacınız olmadığı gibi, TCC zaten C++ derleyicisi olmadığı için her şeyi doğru yapsanız bile cpp derleyemezsiniz.

Vista’da (32-bit Windows Vista Home Premium ve Ultimate Edition’larda denemiştir.) Code Blocks kullanarak C# ve C++ projeleri şöyle derlenir:

  1. Code Blocks’un sitesine gidin.
  2. Son sürümü (MinGW ile beraber) indirin. (bu yazı yazılırken [codeblocks-8.02mingw-setup.exe](http://garr.dl.sourceforge.net/sourceforge/codeblocks/codeblocks-8.02mingw-setup.exe) olur kendisi.)
  3. Kurun.

Bitti. Artık Vista’da CodeBlocks’u kurmak bu kadar basit. Adamlar sonunda doğrudan çalışan bir sürüm yapmışlar. =)

İyi kodlamalar. Nesne yönelimli günler diliyorum. =)

Not: Bu yöntemin çalışması için CodeBlocks'un önceki sürümlerine ait ayarlarını silmiş olmanız gerekir. Vista'da ayarlar aşağıdaki klasörde depolanır: C:\Users\[KULLANICI ADINIZ]\AppData\Roaming\codeblocks\ Bu klasörü ve yüklüyse CodeBlocks'un önceki tüm sürümlerini/derleyicilerini silmelisiniz.
